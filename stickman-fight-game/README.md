# 🔥 火柴人打斗 (Stickman Fight)

物理碰撞的火柴人打斗游戏，与 AI 对手进行格斗。

## 🎮 游戏特性

- **物理碰撞**: 真实的物理碰撞系统
- **打斗动画**: 流畅的打斗动画
- **AI 对手**: 智能 AI 对手
- **难度等级**: 多个难度选择

## 🚀 快速开始

```bash
npm install
npm run dev
```

## 📖 游戏说明

- **移动**: 方向键或 WASD
- **攻击**: 空格键或点击
- **目标**: 击败 AI 对手

## 🛠️ 技术栈

- React 18 + TypeScript
- Canvas API
- TailwindCSS
