# 🎯 跳一跳 (Jump Game)

物理引擎驱动的跳跃冒险游戏。

## 🎮 游戏特性

- **物理引擎**: 真实的重力和速度系统
- **平台生成**: 随机生成不同高度的平台
- **高度统计**: 实时显示当前高度
- **得分系统**: 根据高度计算得分
- **碰撞检测**: 精确的平台碰撞判定

## 🚀 快速开始

```bash
npm install
npm run dev
```

## 📖 游戏说明

1. 按住鼠标/触屏蓄力
2. 松开跳跃，力度越大跳得越高
3. 在平台上着陆继续跳跃
4. 掉下去游戏结束
5. 尽可能跳得更高

## 🛠️ 技术栈

- React 18 + TypeScript
- Canvas API
- 物理引擎

## 🎯 游戏规则

- 蓝色方块 = 玩家
- 绿色平台 = 可着陆平台
- 红色平台 = 危险区域
