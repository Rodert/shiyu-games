# 🐍 贪吃蛇 (Snake Game)

经典的贪吃蛇游戏，操控蛇吃食物并避免碰撞。

## 🎮 游戏特性

- **经典玩法**: 原汁原味的贪吃蛇游戏机制
- **动态难度**: 随分数增加难度提升
- **霓虹效果**: 炫酷的霓虹灯视觉效果
- **实时统计**: 显示分数、长度等信息
- **暂停功能**: 支持暂停和继续游戏

## 🚀 快速开始

```bash
npm install
npm run dev
```

## 📖 游戏说明

### 基本操作
- **移动**: 方向键或 WASD
- **暂停**: P 键
- **目标**: 吃食物增长，避免碰撞

### 游戏规则
1. 蛇在游戏区域内移动
2. 吃到食物蛇身增长 1 节
3. 每吃一个食物获得 10 分
4. 碰到墙壁或自己游戏结束
5. 分数越高难度越大

### 难度系统
- **初级**: 蛇速度较慢
- **中级**: 蛇速度中等
- **高级**: 蛇速度很快

## 🛠️ 技术栈

- React 18 + TypeScript
- Canvas API
- TailwindCSS
