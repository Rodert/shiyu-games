# 🎨 炫彩 2048 (2048 Game)

经典的 2048 滑动合并游戏，通过滑动方块进行合并，目标是达到 2048。

## 🎮 游戏特性

- **经典玩法**: 原汁原味的 2048 游戏机制
- **分数系统**: 实时分数计算和最高分记录
- **撤销功能**: 支持悔棋操作
- **响应式设计**: 完美适配各种设备
- **炫彩视觉**: 彩色方块和流畅动画

## 🚀 快速开始

```bash
npm install
npm run dev
```

## 📖 游戏说明

### 基本操作
- **方向键**: 上下左右移动方块
- **WASD**: 替代方向键
- **触屏**: 滑动屏幕移动

### 游戏规则
1. 每次移动，所有方块向指定方向移动
2. 相同数字的方块碰撞时合并
3. 每次移动后随机生成一个新方块
4. 无法继续移动时游戏结束
5. 达到 2048 可以继续游戏
