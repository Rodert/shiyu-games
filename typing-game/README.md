# ⌨️ 前端打字机 (Typing Game)

极限打字挑战游戏，在限定时间内尽快输入单词。

## 🎮 游戏特性

- **限时挑战**: 60 秒打字竞赛
- **速度测试**: 计算 WPM (Words Per Minute)
- **准确率统计**: 显示打字准确率
- **难度递增**: 单词难度逐级提升

## 🚀 快速开始

```bash
npm install
npm run dev
```

## 📖 游戏说明

- **输入**: 在输入框中输入显示的单词
- **提交**: 按 Enter 键提交
- **目标**: 在 60 秒内输入尽可能多的单词

## 🛠️ 技术栈

- React 18 + TypeScript
- TailwindCSS
