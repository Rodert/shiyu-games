# 😀 Emoji 连连看 (Emoji Match)

配对相同 Emoji 的记忆卡片游戏。

## 🎮 游戏特性

- **12对卡片**: 24张卡片需要配对
- **计时系统**: 记录完成时间
- **步数统计**: 记录点击次数
- **配对进度**: 显示已配对数量

## 🚀 快速开始

```bash
npm install
npm run dev
```

## 📖 游戏说明

1. 点击卡片翻开 Emoji
2. 找到相同的 Emoji 配对
3. 配对成功卡片保持打开
4. 配对所有卡片即可通关
5. 步数越少分数越高

## 🛠️ 技术栈

- React 18 + TypeScript
- TailwindCSS
