# 🎨 色差辨识 (Color Difference)

考验你的色彩辨识能力，找出与其他方块不同颜色的方块。

## 🎮 游戏特性

- **色彩辨识**: 快速识别不同颜色
- **难度递增**: 颜色差异逐渐减小
- **计时系统**: 限时挑战
- **得分系统**: 正确识别获得分数

## 🚀 快速开始

```bash
npm install
npm run dev
```

## 📖 游戏说明

- **点击**: 点击不同颜色的方块
- **目标**: 在时间内找出颜色差异

## 🛠️ 技术栈

- React 18 + TypeScript
- TailwindCSS
