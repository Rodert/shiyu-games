# 🏃 前端跑酷 (Parkour)

刺激的跑酷游戏，跳跃躲避障碍物，不断前进。

## 🎮 游戏特性

- **跳跃躲避**: 跳过障碍物
- **无限模式**: 不断增加难度
- **得分系统**: 躲避障碍获得分数
- **难度递增**: 随分数增加速度提升

## 🚀 快速开始

```bash
npm install
npm run dev
```

## 📖 游戏说明

- **跳跃**: 空格键或点击
- **目标**: 躲避障碍物，获得高分

## 🛠️ 技术栈

- React 18 + TypeScript
- Canvas API
- TailwindCSS
