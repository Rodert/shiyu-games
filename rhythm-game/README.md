# 🎵 节奏点点 (Rhythm Game)

音乐节奏游戏，跟随音乐节奏点击屏幕。

## 🎮 游戏特性

- **音乐节奏**: 跟随音乐节奏
- **精准判定**: Perfect/Good/Miss 判定
- **连击系统**: 保持连击获得奖励
- **得分系统**: 精准度越高分数越多

## 🚀 快速开始

```bash
npm install
npm run dev
```

## 📖 游戏说明

- **点击**: 在正确时间点击屏幕
- **判定**: Perfect/Good/Miss 三级判定
- **目标**: 获得高分和高连击

## 🛠️ 技术栈

- React 18 + TypeScript
- Canvas API
- TailwindCSS
