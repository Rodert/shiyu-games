# 🛡️ 塔防守卫 (Tower Defense)

迷你塔防游戏，放置防御塔抵挡敌人。

## 🎮 游戏特性

- **塔防机制**: 放置防御塔防守
- **波次敌人**: 分波次生成敌人
- **升级系统**: 升级防御塔能力
- **得分系统**: 击败敌人获得分数

## 🚀 快速开始

```bash
npm install
npm run dev
```

## 📖 游戏说明

- **放置**: 点击放置防御塔
- **升级**: 升级防御塔能力
- **目标**: 抵挡所有敌人

## 🛠️ 技术栈

- React 18 + TypeScript
- Canvas API
- TailwindCSS
