# 🎮 仕宇游戏合集 - 31 个游戏完整计划

## 项目概述

将仕宇游戏合集从 9 个游戏扩展到 **31 个游戏**，包括策略、动作、音乐、射击等多个类别。

## 技术栈

### 前端框架
- **React 18** - UI 框架
- **TypeScript** - 类型检查
- **Canvas API** - 游戏渲染
- **DOM** - 界面交互

### 开发工具
- **Vite 5.0** - 构建工具
- **TailwindCSS 3.3** - 样式框架
- **Lucide React** - 图标库

### 部署
- **GitHub Actions** - CI/CD
- **GitHub Pages** - 部署平台
