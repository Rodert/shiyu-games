# 🔵 五子棋人机对战 (Gomoku)

与 AI 对手进行的五子棋游戏，在 15×15 棋盘上先连成五个相同棋子的一方获胜。

## 🎮 游戏特性

- **AI 对手**: 智能 AI 对手
- **15×15 棋盘**: 经典五子棋规则
- **难度等级**: 多个难度选择
- **得分追踪**: 记录胜负统计

## 🚀 快速开始

```bash
npm install
npm run dev
```

## 📖 游戏说明

- **点击**: 在棋盘上点击下棋
- **规则**: 先连成五个相同棋子获胜
- **难度**: 选择 AI 难度等级

## 🛠️ 技术栈

- React 18 + TypeScript
- AI 算法
- TailwindCSS
