# 🔷 几何冲刺 (Geometry Dash)

快速反应的几何障碍躲避游戏，跟随音乐节奏进行冒险。

## 🎮 游戏特性

- **几何障碍**: 各种几何形状的障碍物
- **音乐同步**: 与音乐节奏同步
- **关卡设计**: 精心设计的关卡
- **难度递增**: 逐级提升的挑战

## 🚀 快速开始

```bash
npm install
npm run dev
```

## 📖 游戏说明

- **跳跃**: 空格键或点击
- **目标**: 躲避障碍物，到达终点

## 🛠️ 技术栈

- React 18 + TypeScript
- Canvas API
- TailwindCSS
