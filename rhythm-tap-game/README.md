# 🎵 节奏点点 (Rhythm Tap)

一个充满节奏感的音乐节奏游戏，考验你的反应速度和节奏感。

## 🎮 游戏特性

- **4 车道节奏系统**: 4 条独立的音乐轨道
- **精准判定系统**: Perfect/Good/Miss 三级判定
- **连击系统**: 保持连击获得更高分数
- **实时反馈**: 即时的分数和统计显示
- **炫酷视觉效果**: Canvas 渲染的发光效果
- **响应式设计**: 完美适配各种设备

## 📖 游戏说明

### 基本操作
- **点击**: 在对应的车道点击音符
- **键盘**: 按 D、F、J、K 键对应 4 条车道
- **鼠标**: 点击游戏区域对应位置

### 游戏规则
1. 游戏开始时音符从上方下落
2. 当音符到达底部的命中区域时点击
3. 根据精准度获得不同分数:
   - **Perfect** (±50ms): 100 分
   - **Good** (±150ms): 50 分
   - **Miss**: 0 分，连击清零
4. 保持连击可以获得更高的最大连击数

### 难度系统
- **无等级系统**: 难度随时间递增
- **音符密度**: 音符生成间隔固定 500ms
- **随机分布**: 音符在 4 条车道随机分布

## 🎯 游戏机制

### 判定系统
```
Perfect: 时间差 < 50ms   → 100分
Good:    时间差 < 150ms  → 50分
Miss:    时间差 > 150ms  → 0分，连击清零
```

### 分数计算
- **Perfect**: 100 分 + 连击奖励
- **Good**: 50 分 + 连击奖励
- **Miss**: 0 分，连击重置为 0

### 连击系统
- 每次成功判定连击 +1
- Miss 时连击重置为 0
- 最高连击数记录保存

## 🔧 配置选项

### 游戏参数 (可在 hook 中调整)
```typescript
// 车道数
LANES: 4

// 车道宽度
LANE_WIDTH: 150

// 音符大小
NOTE_SIZE: 40

// 命中区域 Y 坐标
HIT_ZONE_Y: CANVAS_HEIGHT - 100

// 音符生成间隔
spawnInterval: 500

// 音符下落速度
noteY = HIT_ZONE_Y - (currentTime - note.time) * 0.3

// Perfect 判定范围
timeDiff < 50

// Good 判定范围
timeDiff < 150
```
