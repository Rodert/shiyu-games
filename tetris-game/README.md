# 🧱 俄罗斯方块 (Tetris)

经典的俄罗斯方块游戏，通过旋转和移动方块消除行。

## 🎮 游戏特性

- **经典玩法**: 原汁原味的俄罗斯方块游戏机制
- **7 种方块**: 7 种经典方块形状
- **旋转功能**: 支持方块旋转
- **等级递增**: 随等级提升速度加快
- **消除统计**: 显示消除行数和分数

## 🚀 快速开始

```bash
npm install
npm run dev
```

## 📖 游戏说明

### 基本操作
- **左右移动**: 方向键左右或 A/D
- **旋转**: 方向键上或 W
- **加速下落**: 方向键下或 S
- **暂停**: P 键

### 游戏规则
1. 方块从顶部下落
2. 使用方向键移动和旋转方块
3. 填满一行时该行消除
4. 消除行数越多分数越高
5. 方块堆到顶部游戏结束

### 等级系统
- 消除行数越多等级越高
- 等级越高方块下落速度越快
- 难度逐级递增
