# 🃏 卡牌翻翻看 (Memory Card)

经典的记忆卡牌游戏，翻开卡牌找出相同的配对。

## 🎮 游戏特性

- **记忆挑战**: 考验记忆力
- **配对消除**: 找出相同卡牌消除
- **计时系统**: 记录完成时间
- **分数统计**: 显示翻开次数和分数

## 🚀 快速开始

```bash
npm install
npm run dev
```

## 📖 游戏说明

- **点击**: 点击卡牌翻开
- **配对**: 找出两张相同的卡牌
- **目标**: 消除所有卡牌

## 🛠️ 技术栈

- React 18 + TypeScript
- TailwindCSS
