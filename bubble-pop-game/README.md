# 💨 气泡爆破 (Bubble Pop)

点击气泡爆破的快速反应游戏。

## 🎮 游戏特性

- **气泡物理**: 气泡在屏幕内移动
- **连击系统**: 连续爆破获得奖励
- **60秒计时**: 限时挑战
- **得分系统**: 连击倍数加分

## 🚀 快速开始

```bash
npm install
npm run dev
```

## 📖 游戏说明

1. 点击开始游戏
2. 快速点击气泡爆破
3. 连续爆破获得连击奖励
4. 60秒内尽可能多地爆破气泡
5. 暂停/继续游戏

## 🛠️ 技术栈

- React 18 + TypeScript
- Canvas API
- TailwindCSS
