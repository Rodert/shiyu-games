# ⚡ 反应力测试 (Reaction Test)

测试你的反应速度的游戏。

## 🎮 游戏特性

- **反应时间测量**: 精确测量毫秒级反应速度
- **5轮测试**: 完成5轮得出平均反应时间
- **最快反应**: 显示最快反应时间
- **平均反应**: 显示平均反应时间
- **统计分析**: 显示所有反应时间

## 🚀 快速开始

```bash
npm install
npm run dev
```

## 📖 游戏说明

1. 点击开始测试
2. 等待屏幕变绿
3. 尽快点击绿色区域
4. 完成5轮测试
5. 查看你的反应速度统计

## 🛠️ 技术栈

- React 18 + TypeScript
- TailwindCSS

## 📊 反应速度标准

- **< 200ms**: 超人反应
- **200-300ms**: 优秀
- **300-400ms**: 良好
- **> 400ms**: 需要训练
