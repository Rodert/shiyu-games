# 🌀 迷宫生成器 (Maze Game)

一个使用递归回溯算法生成随机迷宫的冒险游戏。

## 🎮 游戏特性

- **随机迷宫生成**: 使用递归回溯算法生成独特的迷宫
- **多难度等级**: 简单、中等、困难三个难度
- **计时系统**: 记录完成迷宫所需时间
- **得分统计**: 根据难度和时间计算得分
- **暂停/继续**: 随时暂停游戏

## 🚀 快速开始

```bash
npm install
npm run dev
```

## 📖 游戏说明

1. 选择难度等级
2. 使用方向键移动玩家（绿色方块）
3. 到达目标（红色方块）完成关卡
4. 按 P 暂停/继续游戏
5. 按 R 重新开始

## 🛠️ 技术栈

- React 18 + TypeScript
- Canvas API
- TailwindCSS

## 📊 核心算法

**递归回溯迷宫生成**:
- 从起点开始随机选择方向
- 递归访问相邻未访问的单元格
- 回溯处理死路

## 🎯 游戏规则

- 绿色方块 = 玩家
- 红色方块 = 目标
- 白色线条 = 墙壁
- 黑色区域 = 可通行区域
