# 🔨 打地鼠 (Whack Mole)

经典的打地鼠游戏，快速点击出现的地鼠获得分数。

## 🎮 游戏特性

- **反应速度测试**: 考验你的反应速度
- **计时系统**: 限时挑战
- **难度递增**: 地鼠出现速度逐级提升
- **得分系统**: 击中地鼠获得分数

## 🚀 快速开始

```bash
npm install
npm run dev
```

## 📖 游戏说明

- **点击**: 快速点击出现的地鼠
- **目标**: 在限定时间内击中尽可能多的地鼠

## 🛠️ 技术栈

- React 18 + TypeScript
- TailwindCSS
