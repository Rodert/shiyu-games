# 🏃 Flappy 跑酷 (Flappy Parkour)

一个充满挑战的跑酷游戏，结合了 Flappy Bird 的跳跃机制和跑酷的障碍躲避。

## 🎮 游戏特性

- **物理引擎**: 真实的重力和跳跃力学
- **动态障碍**: 随机生成的障碍物和间隙
- **分数系统**: 通过障碍物获得分数
- **最高分记录**: 本地存储最高分
- **霓虹视觉效果**: 炫酷的 Canvas 渲染
- **响应式设计**: 完美适配各种设备

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 📖 游戏说明

### 基本操作
- **跳跃**: 按空格键或点击"跳跃"按钮
- **目标**: 躲避红色障碍物，通过间隙获得分数
- **游戏结束**: 碰到障碍物或屏幕边界时游戏结束

### 游戏规则
1. 游戏开始时玩家处于屏幕左侧中间
2. 按空格键使玩家跳跃
3. 躲避迎面而来的红色障碍物
4. 成功通过障碍物间隙获得 1 分
5. 碰到障碍物或超出屏幕边界游戏结束

### 难度递增
- 游戏没有等级系统，但分数越高难度越大
- 障碍物间隙随机生成，增加不可预测性
- 玩家需要快速反应和精准操作

## 🎯 游戏机制

### 物理系统
- **重力**: 每帧增加向下速度 (0.6 像素/帧²)
- **跳跃**: 按键时速度设置为 -12 像素/帧
- **碰撞检测**: 与障碍物和屏幕边界的碰撞判定

### 障碍物生成
- **生成间隔**: 每 2000ms 生成一个新障碍物
- **移动速度**: 每帧向左移动 5 像素
- **间隙大小**: 150 像素高度
- **随机位置**: 间隙高度随机生成

### 分数计算
- **通过障碍物**: +1 分
- **最高分**: 自动保存到本地存储
