# ✋ 石头剪刀布 (Rock Paper Scissors)

与 AI 对手进行的经典石头剪刀布游戏。

## 🎮 游戏特性

- **随机 AI**: AI 随机选择
- **胜率统计**: 显示胜率百分比
- **游戏历史**: 记录最近10局
- **实时统计**: 得分、胜负、平局

## 🚀 快速开始

```bash
npm install
npm run dev
```

## 📖 游戏说明

1. 选择你的手势（石头、布、剪刀）
2. AI 随机选择
3. 比较结果判断胜负
4. 石头胜剪刀，剪刀胜布，布胜石头
5. 相同手势为平局

## 🛠️ 技术栈

- React 18 + TypeScript
- TailwindCSS
