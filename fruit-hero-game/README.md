# 🍓 水果英雄跑酷 (Fruit Hero)

操控水果角色进行跑酷冒险，收集水果并躲避障碍。

## 🎮 游戏特性

- **水果角色**: 可爱的水果角色
- **收集系统**: 收集水果获得分数
- **躲避机制**: 躲避障碍物
- **难度递增**: 随分数增加难度提升

## 🚀 快速开始

```bash
npm install
npm run dev
```

## 📖 游戏说明

- **移动**: 方向键或 WASD
- **跳跃**: 空格键
- **目标**: 收集水果，躲避障碍

## 🛠️ 技术栈

- React 18 + TypeScript
- Canvas API
- TailwindCSS
