# ⭕ 井字棋 AI 对战 (Tic Tac Toe)

与 AI 对手进行的经典井字棋游戏，采用 Minimax 算法。

## 🎮 游戏特性

- **AI 对手**: Minimax 算法实现的智能 AI
- **三难度等级**: 简单、中等、困难
- **得分追踪**: 记录胜负统计
- **难度调整**: 随时切换难度

## 🚀 快速开始

```bash
npm install
npm run dev
```

## 📖 游戏说明

1. 选择难度等级
2. 你是 ✕，AI 是 ◯
3. 点击空位下棋
4. 先连成三个相同符号的一方获胜
5. 难度越高 AI 越聪明

## 🛠️ 技术栈

- React 18 + TypeScript
- Minimax 算法
- TailwindCSS

## 🎯 AI 难度

- **简单**: AI 随机移动
- **中等**: AI 70% 最优移动
- **困难**: AI 总是最优移动
