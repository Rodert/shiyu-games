# ✈️ 飞机大战 (Airplane Game)

经典的飞机射击游戏，操控飞机躲避敌机并发射子弹消灭敌人。

## 🎮 游戏特性

- **飞机控制**: 流畅的飞机操作
- **敌机生成**: 随机生成敌机
- **子弹系统**: 连续射击敌机
- **得分系统**: 消灭敌机获得分数
- **难度递增**: 随着分数增加难度提升

## 🚀 快速开始

```bash
npm install
npm run dev
```

## 📖 游戏说明

- **移动**: 方向键或 WASD
- **射击**: 空格键
- **目标**: 消灭敌机，避免碰撞

## 🛠️ 技术栈

- React 18 + TypeScript
- Canvas API
- TailwindCSS
