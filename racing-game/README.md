# 🏎️ 前端赛车 (Racing Game)

2D 赛车游戏，操控赛车在赛道上竞速，支持漂移和加速。

## 🎮 游戏特性

- **赛车控制**: 流畅的赛车操作
- **漂移系统**: 支持漂移加速
- **计时系统**: 记录完成时间
- **难度递增**: 赛道难度逐级提升

## 🚀 快速开始

```bash
npm install
npm run dev
```

## 📖 游戏说明

- **移动**: 方向键或 WASD
- **漂移**: 空格键
- **目标**: 完成赛道，获得最快时间

## 🛠️ 技术栈

- React 18 + TypeScript
- Canvas API
- TailwindCSS
