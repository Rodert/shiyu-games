# 🔢 数字华容道 (Sliding Puzzle)

经典的 15 拼图游戏，通过滑动方块排列成正确顺序。

## 🎮 游戏特性

- **15 拼图**: 经典的 4×4 拼图
- **计时系统**: 记录完成时间
- **移动计数**: 统计移动次数
- **难度选择**: 多个难度等级

## 🚀 快速开始

```bash
npm install
npm run dev
```

## 📖 游戏说明

- **点击**: 点击要移动的方块
- **目标**: 将方块排列成 1-15 的顺序
- **规则**: 只能移动与空位相邻的方块

## 🛠️ 技术栈

- React 18 + TypeScript
- TailwindCSS
