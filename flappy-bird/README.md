# 🐦 像素飞鸟 (Flappy Bird)

经典的 Flappy Bird 游戏复刻，通过点击或按键使小鸟上升，躲避管道。

## 🎮 游戏特性

- **经典玩法**: 原汁原味的 Flappy Bird 游戏机制
- **管道躲避**: 躲避上下管道
- **分数系统**: 通过管道获得分数
- **最高分记录**: 本地存储最高分
- **炫彩视觉**: 像素风格的视觉效果

## 🚀 快速开始

```bash
npm install
npm run dev
```

## 📖 游戏说明

### 基本操作
- **跳跃**: 按空格键或点击屏幕
- **目标**: 躲避管道，获得高分

### 游戏规则
1. 小鸟不断下落
2. 按空格键使小鸟上升
3. 躲避上下管道
4. 通过管道间隙获得 1 分
5. 碰到管道或屏幕边界游戏结束

## 🛠️ 技术栈

- React 18 + TypeScript
- Canvas API
- TailwindCSS
