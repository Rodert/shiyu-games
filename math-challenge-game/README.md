# 🧮 算力冲刺 (Math Challenge)

快速数学计算挑战游戏。

## 🎮 游戏特性

- **数学题生成**: 随机生成加减乘除题目
- **三难度等级**: 简单、中等、困难
- **60秒计时**: 限时答题
- **正确率统计**: 显示正确率百分比
- **得分系统**: 正确加分，错误扣分

## 🚀 快速开始

```bash
npm install
npm run dev
```

## 📖 游戏说明

1. 选择难度等级
2. 看题目输入答案
3. 按 Enter 提交答案
4. 60秒内尽可能多地答对题目
5. 难度越高题目越复杂

## 🛠️ 技术栈

- React 18 + TypeScript
- TailwindCSS
