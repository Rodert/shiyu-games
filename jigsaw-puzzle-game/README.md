# 🧩 在线拼图 (Jigsaw Puzzle)

经典的拼图游戏，选择难度后拼完整图片。

## 🎮 游戏特性

- **难度选择**: 多个难度等级
- **随机图片**: 每次随机不同图片
- **计时系统**: 记录完成时间
- **拖拽操作**: 直观的拖拽拼图

## 🚀 快速开始

```bash
npm install
npm run dev
```

## 📖 游戏说明

- **拖拽**: 拖动拼图块到正确位置
- **难度**: 选择拼图块数量
- **目标**: 完成整个拼图

## 🛠️ 技术栈

- React 18 + TypeScript
- Canvas API
- TailwindCSS
