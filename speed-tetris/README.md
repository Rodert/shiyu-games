# ⚡ 极速方块 (Speed Tetris)

加速版的俄罗斯方块，方块下落速度极快，考验你的反应速度。

## 🎮 游戏特性

- **加速下落**: 方块下落速度极快
- **连击系统**: 连续消除获得奖励
- **难度递增**: 随等级提升速度加快
- **得分系统**: 消除行数获得分数

## 🚀 快速开始

```bash
npm install
npm run dev
```

## 📖 游戏说明

- **移动**: 方向键或 WASD
- **旋转**: 方向键上或 W
- **目标**: 在极快的速度下消除行

## 🛠️ 技术栈

- React 18 + TypeScript
- Canvas API
- TailwindCSS
