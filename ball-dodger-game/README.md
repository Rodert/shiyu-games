# 🔵 小球躲避器 (Ball Dodger)

无限生存模式的躲避游戏，躲避不断增多的球体。

## 🎮 游戏特性

- **无限模式**: 不断增加难度
- **躲避机制**: 躲避球体获得分数
- **难度递增**: 球体数量和速度递增
- **实时统计**: 显示当前分数和最高分

## 🚀 快速开始

```bash
npm install
npm run dev
```

## 📖 游戏说明

- **移动**: 鼠标移动或方向键
- **目标**: 躲避球体，生存尽可能长时间

## 🛠️ 技术栈

- React 18 + TypeScript
- Canvas API
- TailwindCSS
