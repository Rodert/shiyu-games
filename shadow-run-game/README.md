# 🌙 光影酷跑 (Shadow Run)

夜间特效的跑酷游戏，光影效果营造紧张氛围。

## 🎮 游戏特性

- **光影效果**: 炫酷的光影视觉效果
- **夜间主题**: 黑暗中的冒险
- **无限模式**: 不断增加难度
- **得分系统**: 躲避障碍获得分数

## 🚀 快速开始

```bash
npm install
npm run dev
```

## 📖 游戏说明

- **移动**: 方向键或 WASD
- **目标**: 躲避障碍物，获得高分

## 🛠️ 技术栈

- React 18 + TypeScript
- Canvas API
- TailwindCSS
